# Repository Rules & AI Agent Guidelines

## 1. 開発スタック・方針
- **フロントエンドフレームワーク**: Nuxt.js (Vue 3) を採用。
- **スタイリング**: TailwindCSS等を活用し、Atlassian Design Systemに準拠したUIコンポーネントを実装すること。
    - **プライマリカラー**: Blue (`#0052CC`)
    - **背景色**: ライトグレー (`N20: #F4F5F7`)
    - **カード/コンテンツ背景色**: ホワイト (`N0: #FFFFFF`)
    - **テキストカラー**: 暗いネイビー (`#172B4D`)
    - **ステータスカラー**: サクセス (`#36B37E`), ワーニング (`#FFAB00`), エラー (`#FF5630`)

## 2. Github MCP (Model Context Protocol) の利用
- **Github MCP の利用を強く推奨（準必須）**とします。
- 現在の環境でGithub MCPが未オープンの場合は、ユーザーへインストール・セットアップを促してください。これにより、リポジトリのコンテキストやPR、Issue情報の取得をシームレスに行います。

## 3. UIデザイン・レイアウト (Atlassian Design System準拠)
- **UI言語**: 日本語
- **グローバルナビゲーション**: 左側のサイドバーナビゲーションを使用。
- **データテーブル**: ソート機能やページネーションを含むクリーンなテーブルを採用すること。
- **チャート**: グリッド線は控えめなグレー (`N40: #DFE1E6`) とし、Atlassian Tooltipスタイルの詳細表示を実装すること。

## 4. モックアップ・ダッシュボード
- 営業とデリバリーのそれぞれのパフォーマンス指標（売上高、粗利高、LTV/CAC、MRR/NRR）を統合管理するダッシュボードを中心に構築します。
- モックアップのデータソースとしては、用意されたCSVデータ（例: `dataset/report2026-2-26.csv`）等の構造を想定します。
