# Kintone 画像一括ダウンロードプラグイン

レコードの作成期間を指定して、その期間内に作成されたレコードに含まれる画像を一括でダウンロードできるKintoneプラグインです。

## 機能

- レコードの作成期間（開始日・終了日）を指定
- 指定期間内のレコードを自動取得
- レコード内の画像フィールド（ファイルフィールド）を自動検出
- 画像をZIPファイルとして一括ダウンロード
- ダウンロード時の進捗表示

## インストール方法

### 1. プラグインのパッケージ化

```bash
npm install
npm run package
```

または、[kintone-plugin-packer](https://github.com/kintone/js-sdk/tree/master/packages/plugin-packer)を使用してプラグインをパッケージ化します。

### 2. Kintoneにプラグインをアップロード

1. Kintoneの「設定」→「プラグイン」にアクセス
2. 「プラグインを追加」をクリック
3. パッケージ化されたプラグイン（`.zip`ファイル）をアップロード
4. プラグインを有効化

## 使用方法

1. 画像が含まれるアプリのレコード一覧画面にアクセス
2. ヘッダー部分の「画像一括ダウンロード」ボタンをクリック
3. 開始日と終了日を指定
4. 「ダウンロード開始」ボタンをクリック
5. 処理が完了すると、ZIPファイルが自動的にダウンロードされます

## ダウンロードされるファイル構造

```
kintone_images_YYYY-MM-DD_YYYY-MM-DD.zip
├── フィールド名1/
│   ├── レコード番号_ファイル名1.jpg
│   └── レコード番号_ファイル名2.png
└── フィールド名2/
    └── レコード番号_ファイル名3.jpg
```

## 注意事項

- 大量のレコードがある場合、処理に時間がかかる場合があります
- 画像ファイルは「フィールド名/レコード番号_ファイル名」の構造でZIPに格納されます
- ファイルフィールドのみが対象となります（その他のフィールドタイプは対象外）
- レコードの取得にはKintone REST APIを使用します

## 開発

### ファイル構成

- `manifest.json` - プラグインのマニフェストファイル
- `config.html` - プラグイン設定画面（期間指定UI、現在は未使用）
- `desktop.js` - デスクトップ版のメインロジック
- `icon.png` - プラグインアイコン（同梱済み）

### 必要なライブラリ

- JSZip: ZIPファイル生成に使用（CDNから自動読み込み）
